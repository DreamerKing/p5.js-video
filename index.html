About

Design and development of this demo was lead by Scott Garner with input and contributions from Dan Shiffman and support from NYU ITP.

Development of p5.js is lead by Lauren McCarthy with the support of the Processing Foundation, NYU ITP, Bocoup, and RISD 2050.

Popcorn.js, part of Mozilla's Popcorn project, was used to trigger javascript events in sync with the video. Realtime compositing of the presenters was done with Seriously.js, created by Brian Chirls with support from Mozilla.

Other libraries and frameworks used in this project include the Ace Editor, Twitter Bootstrap and jQuery.
